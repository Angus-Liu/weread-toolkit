## 使用

> 该插件暂时未上架 chrome 网上应用店

1. 下载本项目（若是压缩文件，需解压）
2. 使用 chrome 浏览器打开“管理扩展程序页”（或地址栏输入“chrome://extensions/”直接打开）
3. 打开“开发者模式”
4. 点击“加载已解压的扩展程序”
5. 选择本项目即可
